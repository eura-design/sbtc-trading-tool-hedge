### C5. `useTrendLineAlert.js` — 로그 모드에서 선형 보간으로 알림 트리거
- **위치**: `frontend/src/hooks/useTrendLineAlert.js:6-10` `linePriceAt`
- **현상**: `linePriceAt`는 항상 선형 `p1 + (p2-p1)*(t-t1)/dt`.
  하지만 차트는 로그 모드에서 지수 보간 `p1 * Math.pow(p2/p1, alpha)`로 그려짐 (`components/Chart/TrendLines.jsx:11`, `Channels.jsx`, `utils/hitTest.js:31`).
- **영향**: 가격 범위가 큰 추세선/채널 + 로그 모드 활성 시 알림이 시각적 위치와 다르게 트리거.
- **수정**: `useTrendLineAlert`에 `isLog`는 이미 prop으로 받고 있으나(line 30) `linePriceAt` 호출 시 활용되지 않음. isLog일 때 지수 보간 사용:
  ```js
  function linePriceAt(t1, p1, t2, p2, t, isLog = false) {
    const dt = t2 - t1; if (dt === 0) return p1;
    const a  = (t - t1) / dt;
    if (isLog && p1 > 0 && p2 > 0) return p1 * Math.pow(p2 / p1, a);
    return p1 + (p2 - p1) * a;
  }
  ```
